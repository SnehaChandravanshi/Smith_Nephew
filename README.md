# 📖 Project Overview

This project presents a complete Business Intelligence solution for the **Smith & Nephew Sales, Inventory, and Operations Planning (SIOP) Reporting Case Study**.

The objective was to transform raw operational data into meaningful business insights through data cleaning, validation, transformation, data modeling, and interactive dashboard development using **Power BI**.

The final dashboard enables stakeholders to monitor forecast performance, analyze sales trends, evaluate operational efficiency, and make informed business decisions using interactive visualizations.

---

# 🎯 Project Objectives

- Perform data quality assessment and validation
- Clean and transform raw data using Power Query
- Build a dimensional data model
- Develop business KPIs using DAX
- Create an interactive Power BI dashboard
- Generate actionable business insights
- Document the complete reporting workflow
